# üõí Amazon Product Recommender

Welcome to the **Amazon Product Recommender**! This application helps you find the best products on Amazon based on your preferences. Simply provide a description of what you're looking for, and the app will recommend top products along with AI-powered suggestions to make your shopping experience seamless!

## üîç Features

- **Search for products**: Enter a query like "budget laptop" or "wireless headphones," and get a list of Amazon products that match.
- **AI-powered recommendations**: Using cutting-edge Groq and OpenAI models, the app suggests the best products based on your preferences.
- **User-friendly interface**: Simple and intuitive design with Streamlit for easy interaction.

## üöÄ How to Run the App

1. **Python 3.7+**: Ensure you have Python installed. You can download it from [python.org](https://www.python.org/).
2. **Groq API Key**: You'll need a Groq API key for the AI model. You can obtain it from [Groq](https://groq.com/).

## üß† AI Model Used

The product recommendations are generated using powerful AI models:

* **Groq Llama Model** : Groq's high-performance Llama model is used to process your query and provide AI-driven recommendations.

## üí° How It Works

1. **User Input** : The user provides a query, such as "best budget laptop."
2. **Product Retrieval** : The app fetches relevant products from Amazon using the `retrieve_similar_products` function.
3. **AI Recommendation** : The Groq model (or OpenAI model) analyzes the query and the retrieved products and generates AI-powered recommendations.
4. **Display Results** : The user sees a list of retrieved products and a detailed recommendation with explanations on why certain products were suggested.

## ‚ö°Ô∏è Technologies Used

* **Groq API** : AI-driven product recommendations using Groq's Llama model.
* **OpenAI API** : (Optional) For fallback product recommendations using GPT models.
* **Streamlit** : For creating the interactive web app interface.
* **Python** : Main programming language.
* **dotenv** : For securely managing environment variables.
* **requests** : For making API calls.
